# Polytoria Shirt Tester

3D shirt and clothing tester for **Polytoria avatars**. Upload or link textures, customize body colors, and preview your character in real-time.

## Demo

Hosted via GitHub Pages:

```
https://jvr2022.github.io/polytoria-shirt-tester/
```

## Usage

1. Clone the repo:

```bash
git clone https://github.com/jvr2022/polytoria-shirt-tester.git
```

2. Serve with a local server:

```bash
cd polytoria-shirt-tester
python -m http.server 8080
```

3. Open in browser:

```
http://localhost:8080
```

4. Upload texture or paste URL, select body part, and adjust colors.
--- 
> [!WARNING]
> * Must be served via HTTP(S); `file://` will not work.
> * Self-hosted viewer avoids CORS issues.
